# THREAD & TREND - Luxury Apparel E-Commerce

![Thread & Trend](https://img.shields.io/badge/Thread%20%26%20Trend-Luxury%20Apparel-purple)

## 🛍️ About The Project

Thread & Trend is a premium luxury apparel e-commerce platform inspired by Nike, Zara, and Aime Leon Dore. Built with PHP, MySQL, and Tailwind CSS, it delivers a seamless, Apple-like shopping experience.

## ✨ Features

- **Luxury User Interface** - Premium design with elegant typography and smooth animations
- **Product Management** - Full CRUD operations with image upload
- **Shopping Cart** - Add, edit, delete items with quantity control
- **User Authentication** - Login, Register, Password Reset
- **Admin Dashboard** - Manage products, users, and view audit logs
- **Responsive Design** - Mobile-first, works on all devices
- **Audit Logging** - Track all user activities

## 🚀 Technologies Used

| Technology | Purpose |
|------------|---------|
| PHP 8.2+ | Backend Logic |
| MySQL / MariaDB | Database |
| Tailwind CSS | Styling |

| Playfair Display, Inter | Typography |
